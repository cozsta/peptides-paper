# Peptides — a research reader (RU · EN · ZH)

A detailed, plain-language overview of the peptide / biohacker-compound landscape: what the FDA actually approved vs. the gray market, drug classes (brain / muscle / fat / longevity / sexual), how the Chinese supply chain really works, toxicity & long-term risk, six fact-checked myths, and an honest "what's worth it" verdict.

**Not medical advice.** A map of the territory — decide with your own doctor.

## Read it
Open **`docs/index.html`** and pick a language, or read the source:
- 🇷🇺 [`ru.md`](ru.md) · 🇬🇧 [`en.md`](en.md) · 🇨🇳 [`zh.md`](zh.md)

Live (after GitHub Pages is enabled): `https://cozsta.github.io/peptides-paper/`

## Build
```bash
./build_html.sh        # renders docs/{ru,en,zh}.html + index.html via pandoc
```
Requires `pandoc`. The reader UI (mobile-first, dark mode, drawer table-of-contents, language switcher) lives in `template.html`.

## Sources
FDA/PCAC documents, ClinicalTrials, Falutz NEJM 2007, Nass Ann Intern Med 2008, BMJ 2024 (GLP-1/thyroid cohort), Vilsbøll DOM 2026, Venhuis JMIR 2024, Janoshik public database, TGA 2026, DermNetNZ.

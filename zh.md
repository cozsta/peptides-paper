# 肽类物质：哪些有效，哪些是炒作，哪些危险——以及如何避免在灰色市场上中毒

*本综述基于我们的研究（2026年）。这不是医疗建议——而是一份地图：哪些已获批准，哪些真正经过验证，哪些是神话，以及市场的运作方式。每个人最终要结合自身情况和医生意见做出决定。*

---

## 核心要点（只看这部分也够）

1. **灰色肽类市场 ≈ 约90%的炒作 + 约10%的真实医学。** 真实医学主要是GLP-1（Ozempic及同类药物）和少数获批药物。其他一切都以"研究化学品，非人用"的标签销售——这是**法律漏洞，不是质量保证**。
2. **肽是针对特定问题的工具**，而不是"让自己变得更好的全能药"。有问题（肥胖、久治不愈的创伤、性欲低下、内脏脂肪）→ 有对应的肽。没有问题 → 几乎一切都落入"没必要"。
3. **最大的风险不是分子本身，而是产品。** 奇异的恐惧（肽致癌）几乎全部是理论性的、来自啮齿动物的实验。而内毒素发热、非无菌注射引起的脓肿和瓶内重金属——这些是可量化的、发生在人身上的、有文献记录的真实风险。在灰色市场上，安瓿里的污物会在肽本身伤害你之前先置你于死地。

## 1. 什么是肽（Peptide）

肽是短链氨基酸（2–50个），介于"氨基酸"和"蛋白质"之间。从生物学角度看，它们是**信号分子**：模拟激素或生长因子，激活体内特定受体。

- **注射型**（大多数）——肽键在胃中断裂，因此需皮下注射（SC）或肌肉注射（IM）。
- **口服型**（少数例外）——对消化稳定：MK-677（根本不是肽）、KPV（通过PepT1转运体吸收）、BPC-157（在肠道局部对胃酸稳定）。

为何会有灰色市场：约90%生物黑客感兴趣的肽**未经任何监管机构批准**用于任何常规疾病。它们以"research / not for human use"（研究用/非人用）标签销售。几乎所有"研究级"产品都在中国约3–4家工厂合成，西方"品牌"只是将同样的粉末重新分装。

### 研究级/灰色市场——未获批准（以"研究用"销售）
BPC-157、TB-500、GHK-Cu、KPV、ipamorelin（伊帕瑞林）、CJC-1295、sermorelin（舍莫瑞林）、GHRP-2/6、hexarelin（六肽）、**MK-677**、MOTS-c、humanin（人源素）、**5-Amino-1MQ**、epitalon（依皮肽龙）、thymalin（胸腺素）、thymosin-α1（胸腺素α1）、ARA-290、**melanotan I/II**（黑色素坦I/II）、PT-141（灰色瓶装）、Semax（司美克斯）、Selank（赛兰克）、**Dihexa**（地海克萨）等约200种化合物。

### "又重新合法了"的陷阱（重要——卖家常在这一点撒谎）
- **2023年底：** FDA将约19种流行肽（BPC-157、TB-500、GHK-Cu、KPV……）列入"第2类"——存在重大风险（杂质、免疫原性、缺乏数据）。美国药房的合法配制停止。
- **2026年2月：** 宣布*意向*将约14种从第2类恢复至第1类。但Melanotan II、GHRP-2/6等**仍受限制**。
- **2026年4月：** 正式将其从第2类移除——**原因是申请撤回，而非FDA确认安全性**。
- **核心：**"从第2类移除"≠"可以合法配制"。这需要独立的程序，截至2026年中**尚未发生**。因此，"FDA批准/再次合法"的宣传是**谎言**。
- **GLP-1另说：** 在美国合法配制廉价Ozempic仿制品**已终结**（短缺已解除，法庭驳回诉讼，FDA走向永久禁止仿制品——原因：455+份剂量错误造成伤害的报告）。现在只剩品牌处方药或灰色市场。

## 3. 各类别——机制、剂量、益处、风险

### 🧠 大脑（益智/神经类）
总体现实：对健康大脑的效果微弱，最佳数据来自损伤后恢复（中风、创伤），且很多是俄罗斯临床数据，缺乏西方双盲试验。

- **Semax**（+ N-乙酰基变体）——专注力、记忆力、神经保护。升高BDNF/NGF + 多巴胺，不是兴奋剂。**鼻腔给药**，200–600 mcg/天。作用区域：海马体、前额皮质。俄罗斯临床数据（在那里批准用于中风/认知），西方随机对照试验几乎没有。**🟡 在有一定临床历史的"纯专注"选项中表现最佳。**
- **Selank**（+ 酰胺化变体）——抗焦虑/平静专注而不镇静。BDNF + GABA/脑啡肽。鼻腔给药250–900 mcg/天。**🟡 Semax+Selank组合——进入神经肽领域最合理的切入点。**
- **Bromantane**（溴孟坦）（非肽）——平静能量 + 抗疲劳。升高多巴胺*合成*（无反弹/崩溃）。口服50–100 mg。**🟢/🟡 被低估。**
- **Noopept**（诺普特）——记忆/专注，BDNF，AMPA调节。口服10–30 mg（需配合胆碱，否则头痛）。效果微弱。**🟡 便宜、口服、低风险。**
- **Cerebrolysin（脑活素）/ Cortexin（科特克欣）**——神经再生（中风/颅脑损伤/痴呆），神经营养因子混合物。需IM/IV疗程注射，疼痛，动物来源。**在病理情况下有真实试验**，健康人群无数据。**🟡 用于康复 · ⚪ 用于"变聪明"。**
- **Dihexa**——强效突触生成（HGF/c-Met通路激动剂）。⚠️ 这是**增殖性、潜在促癌通路** + 人类数据为零。**🟡/🔴 高风险实验。**
- **P21 / PE-22-28 / NAP**——神经发生/抗抑郁。啮齿动物数据（NAP在人体试验中失败）。**⚪ 推测性。**
- **DSIP**（δ-睡眠诱导肽）——助眠，数据薄弱 **🟡**。**Methylene blue**（亚甲蓝）——大脑线粒体，⚠️ 与抗抑郁药存在危险相互作用（5-羟色胺综合征）**🟡**。**Phenibut**（苯基布）——这不是益智药，而是GABA激动剂，**有依赖性和严重戒断反应** **🔴**。

### 💪 肌肉/恢复/生长激素
- **HGH（生长激素）及促泌素**（ipamorelin、CJC-1295、sermorelin、GHRP-2/6、hexarelin、MK-677）。所有这些最终都会升高生长激素 → IGF-1（胰岛素样生长因子-1）。
  - **Ipamorelin**——最"纯净"（不升高皮质醇、催乳素、食欲）。
  - **GHRP-6** = 强烈饥饿感；**GHRP-2** = 皮质醇+催乳素升高；**hexarelin** = 皮质醇升高最多 + 快速耐受。
  - **MK-677**（口服）——升高GH/IGF-1，但在2年期试验（Nass 2008）中**所有重要指标均失败**：力量未改善，内脏脂肪未减少，反而+水分潴留，+血糖，+皮质醇，胰岛素敏感性下降。水肿和"脸部浮肿"就源于此。
  - **该类别的诚实结论：** 促泌素的存在是为了**恢复**缺乏生长激素的患者和老年人的激素水平。对年轻健康人，它们会将IGF-1推向高位（参见长寿——存在争议），且**没有一种能升高睾酮**。
- **BPC-157**——促血管生成，修复韧带/肠道。数据：所有"运动"相关的数据来自啮齿动物 + 1篇12名患者的回顾性研究（无对照组）+ 2名患者的试验。口服用于胃肠道问题更合理。**🟡 用于特定创伤/肠道问题。**
- **TB-500**——细胞迁移，促愈合。仅啮齿动物数据（人类随机对照试验仅有角膜）。**⚪/🟡。**
- **GHK-Cu**（铜肽）——**外用（乳霜）**对皮肤/胶原蛋白确实有效，有对照试验。注射/系统性用药——零数据。**🟡 作为护肤品。**
- **Follistatin / ACE-031 / 肌肉生长抑素（Myostatin）阻断剂**——理论上能增肌，但**存在严重风险**（试验中ACE-031出现出血事件）。**🔴 对业余者不推荐。**

### 🔥 脂肪/代谢
- **GLP-1（semaglutide、tirzepatide、retatrutide及新药）**——**这是确实有效、效果强大、具有海量证据基础的类别。** 如果有人问"肽类到底有没有用"——这就是证明。对超重人群——改变人生。
  - ⚠️ 主要注意事项：**减少的体重中25–30%是肌肉**，不只是脂肪。不配合力量训练和高蛋白摄入，体成分质量会下降。此外还有胃肠道副作用，以及停药后体重反弹。
- **Tesamorelin**——唯一**有证据**选择性减少内脏脂肪的药物（-15–19%，体重不变）。但：数据仅来自HIV患者，约30%无应答，停药后**3–6个月内效果消退**，且IGF-1升高+81%。
- **SLU-PP-332、5-Amino-1MQ、MOTS-c**——"运动模拟器"，理论上燃烧脂肪/修复代谢。**仅啮齿动物数据，零人类数据。** 机制看起来漂亮，安全性——完全未知。**⚪ 推测性实验，不可押注。**

### ⏳ 长寿/线粒体
- **NAD+ / NMN / NR**——流行，"能量与年轻"。实际上：口服剂型便宜，且与昂贵点滴的（未经证实的）效果相同。**⚪ NAD+点滴 = 主要是昂贵的安慰剂表演。**
- **Epitalon（依皮肽龙）、thymalin、Khavinson生物调节剂**——俄罗斯老年医学，**所有文献来自同一研究组，西方未能重现**。炒作 >> 数据。**⚪。**
- **Thymosin-α1（胸腺素α1）、SS-31、雷帕霉素（Rapamycin）、衰老细胞清除剂（fisetin/quercetin非瑟素/槲皮素）**——有真实数据，但**针对患病人群**（肝炎、脓毒症、特定疾病），不适用于健康人"延长寿命"。

### ❤️ 性/性欲/美容
- **PT-141（Bremelanotide，布美诺肽）**——**最具证据支持的性欲增强选项。** 通过大脑发挥作用（不像伟哥通过血管）。剂量1.75 mg皮下注射，在~45分钟前**按需使用**。副作用：恶心（~40%），血压升高+心率减慢约12小时（高血压患者禁用），频繁使用可能导致**面部/牙龈持续性色素沉着**。每月使用限制约8次。**🟡 按需用于性生活。**
- **Melanotan II**（黑色素坦II）——用于晒黑和性欲。**🔴 避免使用。** 有文献记录：黑色素瘤（melanoma）、非典型痣爆发、现有痣变深、阴茎异常勃起（priapism）、横纹肌溶解+肾衰竭、肾梗死、神经系统综合征PRES。澳大利亚自2026年2月起将其列为毒品（附表9，与海洛因同级）。对长期暴露于阳光的人尤其危险。
- **GHK-Cu（铜肽）、Argireline（精氨酸）、Matrixyl（棕榈酰五肽）等美容肽**——外用对皮肤/皱纹有温和效果，风险低。**🟡 作为乳霜可以。**

## 4. 中国市场——它真实的运作方式（如果你在购买，这是最有用的内容）

**中国生产全球约68%的肽类原料。** 集中区域：江苏省（约35%，GL Biochem、ChinaPeptides等工厂）、浙江（约25%）、山东（约20%，廉价/批量）。2025年中国肽类进口到美国翻倍至约3.28亿美元。**几乎每一个"美国品牌"都是重新分装的中国原料。**

### 为什么杂质是不可避免的
肽通过SPPS（固相肽合成）方法组装——氨基酸逐一连接。每一步都会有一部分链出现缺陷：缺失残基、截短链、镜像异构体（D-型）。这些"碎片"**结构上与目标分子几乎相同**，因此廉价纯化无法将其去除。这才是"99%纯度"背后的真相。

### 三个质量层级
- **GMP制药级（约25–30%）**——洁净室，全面质控。几乎买不到成品。
- **研究级（约40–45%）**——大多数批次有HPLC（高效液相色谱）检测，内毒素偶尔检测。大多数体面原料属于此类。
- **低端（约25–35%）**——质控最少，**证书是模板填写的**。污染和短量就在这里。

### COA（质量证书）——市场上最容易造假的文件
**2024年Janoshik实验室检测的肽类样品中，约43%未达到声称的纯度。** 典型造假手法：
- 不同批次填写相同数字；
- 可疑的整数（恰好99.0%）；
- 重复使用批号；
- 修改PDF后加新卖家名称；
- 用旧检测结果冒充当前货物。

### 即使诚实的COA也不覆盖的内容
- **内毒素（LPS，脂多糖）**——细菌碎片，耐受灭菌，注射后引起发热/脓毒症。瓶子可以是**无菌的，同时含有内毒素**。需要单独的LAL测试。
- **无菌性**——活菌/真菌 → 脓肿（有文献记录的分枝杆菌病例）。
- **重金属**——常规检测不可见。比利时专家检查：约26%的没收注射液砷/铅含量超标（砷高达10倍）。
- **溶剂（TFA三氟乙酸等工业化学品）**——合成的毒性残留，纯度百分比显示不出来。
- **实际毫克量**——纯度是**百分比，不是重量**。瓶子可以99%纯净但只含3 mg而非声称的10 mg（短重/"fronting"冲前量欺诈）。在590个BPC-157样品的数据集中，剂量偏差高达±85%，还发现有直接分子替换。

### 有经验的人如何验证
标准是独立实验室**Janoshik（斯洛伐克）**。规范：**匿名直接寄送样品**，卖家事先不知道结果，结果进入**公开数据库**（卖家无法编辑）。测试——**种类不同**，需根据需要选择：
- **HPLC** → 纯度（%）
- **质谱（Mass spec）** → 确认是否是正确分子（防替换）
- **定量（Quantification/"ID and amount"）** → 实际毫克数（发现短重——**所有人都会忘记的测试**）
- **无菌 + 内毒素（LAL）** → 注射安全性
价格：HPLC约€80–150，每瓶全套约$660。

### 储存与复溶
干粉（冻干品）——冰箱中可保存一到两年。用**抑菌水**（含苯甲醇——多次穿刺时抑制细菌）复溶，沿瓶壁缓慢注入，旋转而非摇晃，+2–8°C储存约28–30天。**复溶不会对已污染的粉末起到灭菌作用。**

### 典型骗局
短重按全价收费；分子替换；假货/降解品；P图证书；不存在的实验室；"如果包裹被没收免费补发"的套路（有时是真的，有时是拿了钱后声称"包裹被没收"但根本没发货）。

### 实操框架"不中毒、不被骗"
1. 每瓶货物**有罪推定**，直到匿名测试证明无罪。
2. 对所有货物：**质谱（是什么）+ HPLC（纯度）+ 定量（几毫克）**。
3. 注射用品额外加：**内毒素 + 无菌检测**。
4. 匿名样品 → Janoshik → 在公开数据库核实。
5. 复溶时保持清洁无菌操作。
6. **如果你对注射没有经验——注射无菌操作/技术会比肽本身更早害了你。**

## 5. 毒性与长期风险——按风险向量分析

**反直觉：产品比分子更危险。**

**风险向量1——分子本身（主要是理论）：**
- *免疫原性* — 更多来自杂质而非肽本身（FDA将肽类放入第2类时正是指出了这一点）。
- *IGF-1 → 癌症——以其被担心的方式来说是神话。* 流行病学证据薄弱且因器官而异。最强信号来自肢端肥大症患者（多年来生长激素极度升高），这**不是适用的模型**。最纯粹的人类类比——成人生长激素替代疗法：癌症**-31%（保护性效果）**，不是增加。真正的注意事项范围很窄：**活动性/近期癌症患者禁用**（IGF-1可能"喂养"已存在的肿瘤）。
- *GLP-1 → 甲状腺癌——神话。* "黑框"警告来自啮齿动物（人类C细胞上这类受体很少）。145,000人的队列研究：风险未增加。注意事项仅保留给有遗传性甲状腺癌病史（MEN2）的人。
- *GLP-1 → 肌肉流失——真实存在。* 最真实的缺点。
- *Melanotan II → 黑色素瘤（melanoma）* — 有真实人类病例记录。

**风险向量2——产品（更大的实际风险）：** 在线销售的标注"99%"纯度的semaglutide，经检验实际纯度为7.7–14.4%（Venhuis 2024），每瓶均含内毒素阳性结果。没收的货物含溶剂、砷/铅、玻璃/橡胶颗粒。非无菌复溶 → 脓肿。**癌症理论是5–20年后的事，且在啮齿动物身上。内毒素发热、脓肿和重金属——是这周的事，且可量化。**

**风险向量3——未知的未知：** 对于BPC-157、TB-500、CJC、ipamorelin、melanotan、MK-677——**零长期人类试验数据。** 即使是已批准的tesamorelin——也只有52周数据。**"安全"在这里 = "没有数据"，而不是"经证明是安全的"。**

## 7. 总结：什么真正值得考虑

肽在三种情况下是合理的，而不是作为"变得更好的按钮"：
1. **有具体缺陷/疾病** — 肥胖 → GLP-1；内脏脂肪 → tesamorelin；久治不愈的创伤 → BPC/TB-500；性欲低下 → PT-141；皮肤老化 → GHK-Cu乳霜。
2. **你是处于边缘状态的运动员，有意识地承担风险**（生长激素/IGF-1）——这是职业选手的游戏，不是健康之道。
3. **你是为了过程而实验** — 那么每瓶都要检测。

**真正有效的（无废话）：** GLP-1（用于超重）· Tesamorelin（内脏脂肪）· PT-141（性欲）· GHK-Cu乳霜（皮肤，风险~零）· BPC/TB-500（特定创伤，前提是原料纯净）。

**炒作/不值钱：** MOTS-c、5-Amino-1MQ、SLU-PP-332（零人类数据）· epitalon、thymalin、生物调节剂 · NAD+点滴 · 大多数"睾酮助推剂"和"长寿堆叠"。

**危险：** Melanotan I/II（黑色素瘤，尤其在阳光下）· 业余者使用肌肉生长抑素阻断剂 · 给"健康人"用生长激素（推高IGF-1，不比训练本身更能增肌，不升高睾酮）。

**如果你健康且没有具体问题——你很可能不需要肽类。** 基础的事情（睡眠、训练、蛋白质、不吸烟）带来的收益超过一柜子安瓿。如果决定尝试——**最大的风险不是分子，而是瓶里的污物**：在Janoshik的匿名检测 + 无菌操作，比选择任何"流行"肽更能保护你。
